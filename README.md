# Listas

## Lista desordenada

É possível fazer uma lista desordenada colocando os seguintes caracteres antes das palavras: -, * ou +.

### Exemplo

+ A
* B
+ C

## Lista ordenada

Para fazer uma lista ordenada, coloque números.

### Exemplo

1. Primeiro
1. Segundo
1. Terceiro
